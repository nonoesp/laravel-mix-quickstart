# laravel-mix-quickstart
Get up and running with HTML, SCSS, and TypeScript in seconds.

## Usage

```bash
# Install npm dependencies
npm install
# Start building the app and watching for changes
npm run watch
# In a new terminal, start a live http server
npm run live
```
